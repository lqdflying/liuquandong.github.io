二. smitty对话屏幕符号含义

这是一个对话屏幕

* —->此字段必须填写一个合法值
# —->这个字段必须填写数字
/ —->此字段必须填写文件或者是目录
x —->此字段必须填写十六进制数值
？ —->此字段中输入的值不会显示出来
+ —->表示此字段的值可以从一个弹出式的列表或者循环列表中选择

三. smitty快捷键

  • F1/ESC+1 —->获取帮助
  • F2/ESC+2 —->刷新当前画面
  • F3/ESC+3 —->取消并返回上一个画面
  • F4/ESC+4 —->list弹出式菜单选项
  • F5/ESC+5 —->修改某个字段值以后，可以用此快捷键重置为默认值
  • F6/ESC+6 —->查看生成的命令行内容
  • F7/ESC+7 —->编辑某个字段，有时候F7含义是Commit
  • F8/ESC+8 —->保存当前名目到SMIT日志中，并显示当前的快捷路径
  • F9/ESC+9 —->暂时进入到shell，SMIT被挂起
  • F10/ESC+0 —->直接退出SMIT，返回命令行
  • 回车/Enter —->执行

四. SMIT的常用快捷路径

  • smitty errpt —->生成一个错误报告
  • smitty shutdown —->关闭系统
  • smitty storge —->管理存储器和文件系统
  • smitty lvm —->逻辑卷管理
  • smitty fs —->文件系统管理
  • smitty filemgr —->文件和目录管理
  • smitty backsys —->系统备份管理
  • smitty mksysb —->做全部系统备份
  • smitty tcpip —->TCP/IP管理
  • smitty mkhostname —->设置主机名
  • smitty mkinet —->增加一个网络接口
  • smitty chinet，shinet —->修改和显示一个网络接口
  • smitty Rminet —->删除一个网络接口
  • smitty Mkuser —->增加一个新用户
  • smitty Chuser —->修改和显示一个用户的属性
